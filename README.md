# Tic Tac Toe Game

This is a simple implementation of the classic game Tic Tac Toe (Xs and Os) built in Python.



## Features

- **Two Player:** Play against a friend.
- **Interactive:** Input your moves using the keyboard.
- **Winning Condition:** Check for three in a row to determine the winner.
- **Board Display:** Clear visualization of the game board after each move.
- **Error Handling:** Handles invalid inputs gracefully.

## How to Play

1. Run the program.
2. The game starts by prompting Player 1 (X) to make a move.
3. Player 2 (O) responds with their move.
4. Alternating turns continue until a player wins or the game ends in a draw.
5. The game prompts for a rematch at the end.
